# Book Catalog System Application

A book catalog system for managing and organizing your library collection.

## Features

- Browse and search books
- Add, edit, and remove book entries
- Catalog metadata (title, author, ISBN, genre, etc.)
- Simple, intuitive interface

## Getting Started

### Prerequisites

- Node.js 18+

### Installation

1. Clone the repository
2. Install dependencies
3. Run the application

```bash
git clone <repository-url>
cd Book_Catalog_System_Aplication
cd frontend
npm install
npm start
```

The app will be available at `http://localhost:5173`
